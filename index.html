A Twitter clone built from scratch with one other person. The server was made with Apache Spark as the backbone to handle HTTP requests. Baseball Season SimulatorRepository

Senior capstone where I worked with one partner. The goal was to find the least amount of statistics necessary to make an accurate simulation. I developed our databases which were SQLite databases due to their portability across platforms and great read performance since reading from them was our main operation. I also created our original object relational mapping scheme. I eventually migrated the project over to Hibernare once it became more optimal not only for performance but also for managing the data as a whole. I also helped develop the code base to drive thee simulations and eventually implemented Maven to help streamline our build process.
